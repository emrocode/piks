## Piks

Piks utilizes Cloudinary's AI image generation to create spooky and atmospheric backgrounds.

### Built with:

- [Nextjs]
- [Tailwindcss]

### Installation and setup

1. Clone this repo

```bash
gh repo clone emrocode/piks
```

2. Go to folder and install dependencies

```bash
npm install
```

3. Run

```bash
npm run dev
```
